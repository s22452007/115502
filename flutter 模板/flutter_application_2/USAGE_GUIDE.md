# 🚀 快速開始指南

## 登錄頁面使用說明

### 登錄步驟

1. **輸入帳號**
   - 在「帳號」欄位輸入任意文字（例如：admin、user123）
   - 不限制格式，任何非空字串都可以

2. **輸入密碼**
   - 在「密碼」欄位輸入任意文字
   - 點擊眼睛圖示可以顯示/隱藏密碼
   - 不需要符合任何密碼規則

3. **選擇功能**
   - **手寫數字辨識 (0-9)** 按鈕（藍色）
     - 適合辨識手寫數字或印刷體數字
     - 會將圖片轉為 28×28 灰階圖
     - 預測結果為 0-9 之間的數字
   
   - **貓狗圖片辨識 🐱🐶** 按鈕（紫色）
     - 適合辨識貓或狗的照片
     - 會將圖片調整為 64×64 彩色圖
     - 預測結果顯示「這是貓」或「這是狗」

### 登錄驗證規則

✅ **允許登錄的情況：**
- 帳號和密碼都有輸入內容
- 帳號：test，密碼：123
- 帳號：admin，密碼：password
- 任何非空的帳號密碼組合

❌ **不允許登錄的情況：**
- 帳號為空
- 密碼為空
- 兩者都為空

## 功能頁面使用說明

### 手寫數字辨識頁面

**適合的圖片：**
- 白色背景上的黑色數字
- 手寫數字照片
- 印刷體數字截圖
- 建議數字佔圖片主要位置

**操作步驟：**
1. 點擊「拍照」或「相簿」選擇圖片
2. 圖片會顯示在畫面上方
3. 點擊「辨識」按鈕開始處理
4. 結果會顯示在圖片下方（例如：預測結果：7）

**技術說明：**
- 圖片會自動轉為灰階
- 自動偵測並裁切數字區域
- 調整為 28×28 像素
- 使用 MNIST 訓練的 Logistic Regression 模型

### 貓狗圖片辨識頁面

**適合的圖片：**
- 清晰的貓或狗照片
- 主體明確的動物照片
- 建議動物佔圖片較大比例

**操作步驟：**
1. 點擊「拍照」或「相簿」選擇圖片
2. 圖片會顯示在畫面上方
3. 點擊「開始辨識」按鈕
4. 等待處理（會顯示載入動畫）
5. 結果顯示在底部彩色區塊：
   - 「這是貓 (信心度: 85.3%)」
   - 「這是狗 (信心度: 92.7%)」

**技術說明：**
- 圖片調整為 64×64 像素
- 保留 RGB 彩色資訊
- 特徵向量為 12,288 維
- 使用二元分類模型

## 快速測試建議

### 測試數字辨識
1. 使用畫圖軟體或紙筆寫一個數字
2. 拍照或截圖
3. 上傳到應用程式
4. 查看預測結果

### 測試貓狗辨識
1. 從網路下載貓或狗的照片
2. 或拍攝家中寵物照片
3. 上傳到應用程式
4. 查看預測結果和信心度

## 常見問題 (FAQ)

**Q: 為什麼需要登錄？**  
A: 這是應用程式的入口設計，用於展示多功能整合。實際上不進行真實驗證，只需輸入任意帳號密碼即可。

**Q: 可以不登錄直接使用嗎？**  
A: 目前版本必須通過登錄頁面。您可以輸入任意內容即可快速通過。

**Q: 辨識結果不準確怎麼辦？**  
A: 目前使用的是示範模型（特別是貓狗辨識），預測結果僅供參考。若要獲得準確結果，需要替換為訓練好的真實模型。

**Q: 支援哪些平台？**  
A: Android、iOS、Web、Windows、macOS、Linux 全平台支援。

**Q: 可以離線使用嗎？**  
A: 可以！模型檔案內建在應用中，不需要網路連線即可辨識。

**Q: 如何返回登錄頁面？**  
A: 在功能頁面點擊左上角的返回按鈕 (←) 即可返回。

**Q: 圖片會被上傳到伺服器嗎？**  
A: 不會！所有處理都在本地裝置上完成，圖片不會上傳到任何伺服器。

## 開發者提示

### 自訂登錄驗證
如需加入真實驗證邏輯，請修改 `lib/pages/login_page.dart` 中的 `_navigateToPage` 方法。

### 修改 UI 主題
主題色彩定義在 `lib/main.dart` 的 `ThemeData` 中。

### 新增更多辨識功能
1. 創建新的模型類別（參考 `cat_dog_classifier.dart`）
2. 創建新的 UI 頁面（參考 `cat_dog_page.dart`）
3. 在 `login_page.dart` 新增選擇按鈕
4. 在 `main.dart` 載入新模型
